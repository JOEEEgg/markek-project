# market-project (쇼핑몰프로젝트)
## #  목적
1. 국비과정 최종프로젝트
2. 수업시간에 배웠던 내용을 기반으로 프로젝트 구성

## # 개발스킬
- Java8
- Spring ver 4.3.22
- Sql/Oracle
- Tomcat 8.5
- Bootstrap, html, css, Jquery
 

## #개발 툴
- STS
- Oracle Developer


## # 기능구현

##### 1. 회원관리
- 로그인 & 회원가입
- 회원정보수정


##### 2. 주문
- 주문하기 & 전체주문하기
- 장바구니


##### 3. 자유게시판
- 게시판 글(등록, 수정, 삭제, 목록)
- 작성자&글내용 검색기능

##### 4. 검색기능
- 상품명, 상품내용으로 검색
- 카테고리, 가격별 검색
